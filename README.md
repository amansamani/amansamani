## 🏗️ Featured Projects

### 🏋️ Fitlip — Indian Diet Nutrition Engine
> **The Problem:** Generic calorie calculators fail on Indian diets because traditional meals don't map neatly to Western food lists.
* **What I Built:** A Mifflin-St Jeor based BMR/TDEE engine featuring goal-normalization (`lean→lose`, `bulk→gain`), meal-slot budgeting, and a Redis-cached food-template service.
* **Tech Stack:** `React Native` `Node.js` `Express` `MongoDB` `Redis` `Razorpay`
* **Data Layer:** Designed custom macro-tagging logic over 400+ indexed Indian food items.

### 🕵️ Fraud Detection — Finding the Fraction of a Percent
> **The Problem:** Highly imbalanced data means naive ML models miss real fraud entirely by defaulting to "not fraud."
* **What I Built:** Trained an XGBoost model on the PaySim dataset, utilizing **SMOTE** to correct class imbalance. Served predictions via a Flask `/predict` API behind a clean dashboard.
* **Tech Stack:** `Python` `XGBoost` `SMOTE` `Flask`
* **Result:** Achieved a **92% ROC-AUC score**.

### 🖼️ Gallery Toons — Shipped & Live
> **The Problem:** Proving I can take a product from raw code to stable production and scale it for real users.
* **What I Built:** A minimalist, highly responsive wallpaper web app deployed and monitored live.
* **Tech Stack:** `HTML` `CSS` `JavaScript` `Vercel`
* **Metrics:** Active platform with **200+ active users** 🚀 [Live Demo](https://gallery-toons.vercel.app)
