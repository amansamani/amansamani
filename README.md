### Projects — what problem, what I built, what happened



**🏋️ Fitlip — a nutrition engine that accounts for how Indians actually eat**

Generic calorie calculators run one formula and a Western food list that doesn't map to Indian diets. I built a Mifflin-St Jeor based BMR/TDEE engine with goal-normalization (`lean→lose`, `bulk→gain`, `fit→maintain`), per-meal-slot calorie budgeting, and a cached food-template service for fast plan generation — backed by 400+ tagged Indian food items (`high-protein`, `high-carb`, `high-fat`, `balanced`) in properly indexed MongoDB collections.

`React Native` `Node.js` `Express` `MongoDB` `Redis` `Razorpay`



**🕵️ Fraud Detection — finding the fraction of a percent**

Fraudulent transactions are a tiny sliver of all transactions, so naive models miss them entirely. Trained an XGBoost model on the PaySim dataset with SMOTE to correct the class imbalance, served through a Flask `/predict` endpoint behind a dark fintech-style frontend.

`Python` `XGBoost` `SMOTE` `Flask` — **92% ROC-AUC**



**🖼️ Gallery Toons — shipped, not just committed**

Wanted proof I could take something from idea to production and keep it running. A wallpaper web app, live on Vercel.

`HTML` `CSS` `JavaScript` `Vercel` — **200+ active users**, live at [gallery-toons.vercel.app](https://gallery-toons.vercel.app)



**🎬 Sentiment Classifier — NLP on noisy real-world text**

LSTM + Word2Vec model trained on 50K+ movie review records, tuned for precision/recall rather than raw accuracy.

`TensorFlow` `LSTM` `Word2Vec` `NLP`
